# Personalized Greeting

Aplicação em JavaScript que exibe uma saudação personalizada com base no nome do usuário e no horário informado, utilizando lógica condicional e manipulação do DOM.

## 🚀 Funcionalidades
- Entrada de nome e horário pelo usuário
- Exibição dinâmica de saudação (Bom dia, Boa tarde ou Boa noite)
- Atualização do conteúdo na tela sem recarregar a página

## 🛠 Tecnologias
- HTML
- JavaScript
